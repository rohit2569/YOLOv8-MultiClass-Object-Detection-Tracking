# YOLOv8-MultiClass-Object-Detection-Tracking
Real-time multi-class object detection and tracking using YOLOv8 (Google Colab)
## Multi-Class Real-Time Object Detection and Tracking Using YOLOv8

This project aims to develop a real-time object detection and tracking system
using YOLOv8. The implementation is carried out using Google Colab and focuses
on detecting multiple object classes from images and video streams.

### Tools & Technologies
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Google Colab
